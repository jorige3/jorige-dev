# jorige.dev

Personal developer portfolio built with **React**, **Vite**, and **Tailwind CSS**.

## 🚀 Features

* Responsive portfolio
* About Me
* Skills
* Featured Projects
* Contact section
* Modern UI

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS

## 📦 Run Locally

```bash
npm install
npm run dev
```
